Connect with GitHub

Your templates are stored in your GitHub repo. Create a token once and all devices stay in sync automatically.

  1. Go to github.com/settings/tokens/new
  2. Give it a name like TemplateFinder
  3. Under Expiration → select No expiration
  4. Under Select scopes → tick repo
  5. Click Generate token → copy it
  6. Paste below and click Connect
